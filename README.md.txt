# World Layoffs SQL Project

## Overview
This beginner project focuses on cleaning and analyzing global layoffs data using PostgreSQL.

## Key Tasks
- Data Cleaning
- Removing Duplicates
- Standardization
- Handling Missing Values
- Exploratory Data Analysis

## Tools Used
- PostgreSQL

## Dataset
Raw dataset sourced from Alex The Analyst GitHub repository.

## Project Structure
- data/: Contains raw and cleaned datasets
- SQL/: Contains SQL scripts for each step
